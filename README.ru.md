# Polymarket Prediction Bot 🤖

Бот, который автоматически анализирует рынки предсказаний на [Polymarket](https://polymarket.com) и находит выгодные ставки.

## Что такое Polymarket?

Polymarket — это платформа, где люди ставят деньги на исход реальных событий: «Биткоин достигнет $100K?», «Кто победит на выборах?», «Будет ли рецессия?». Цена на рынке отражает мнение толпы о вероятности события. Если толпа ошибается — можно заработать.

## Что делает бот?

Бот работает в цикле каждые 5 минут:

1. **Находит интересные рынки** — выбирает активные рынки с большим объёмом торгов по темам: крипто, политика, экономика, технологии, геополитика. Спорт намеренно исключён.

2. **Собирает информацию** — для каждого рынка одновременно ищет:
   - Свежие новости из нескольких источников
   - Движение цены за последний час, день и неделю
   - Кто и сколько ставит (крупные игроки — «киты»)
   - Глубину рынка (сколько денег стоит в очереди на покупку/продажу)
   - Для крипто-рынков — реальные цены монет и индекс страха/жадности

3. **Спрашивает ИИ** — отправляет все собранные данные в Claude (ИИ от Anthropic). Claude оценивает реальную вероятность события и сравнивает с ценой на рынке. Если рынок сильно ошибается — это шанс.

4. **Рассчитывает ставку** — использует формулу Келли (математически оптимальный размер ставки), с поправкой на категорию рынка и уже открытые позиции. Не ставит всё на одну тему.

5. **Отслеживает результаты** — когда рынок закрывается, бот записывает: угадал или нет, сколько заработал/потерял, насколько точны были прогнозы.

## Пример работы

```
Рынок: "Bitcoin выше $90K к 31 марта?"
Цена на Polymarket: 65% (толпа думает — скорее да)

Бот собрал:
  - 5 свежих новостей (ETF приток, регуляция...)
  - BTC вырос на 3% за сутки
  - Крупные игроки ставят на YES (72% conviction)
  - Индекс страха/жадности: 71 (Greed)

Claude оценил: 78% вероятность → edge = 13%
Ставка: $3.20 на YES (Kelly с фракцией 0.25)
```

## Управление через Telegram

Бот присылает уведомления о каждой ставке и результате. Доступные команды:

| Команда | Что делает |
|---------|------------|
| `/stats` | Прибыль, процент побед, ROI |
| `/last` | Последние ставки |
| `/accuracy` | Точность прогнозов по категориям |
| `/config` | Текущие настройки |

## Режимы работы

- **DRY RUN** (по умолчанию) — бот только анализирует и записывает «что бы он поставил». Реальные деньги не тратятся. Полезно для проверки стратегии.
- **LIVE** — реальные ставки. Требует кошелёк и VPN (Polymarket недоступен из некоторых стран).
- **DAILY** — фокус на рынках, которые закрываются сегодня (быстрые ставки).

## Защита от потерь

- Дневной лимит ставок (по умолчанию $25)
- Максимум 10 открытых ставок одновременно
- Максимальная ставка $5
- Не ставит на рынки с маленьким объёмом (мало ликвидности = сложно выйти)
- Не ставит повторно на один рынок в течение 24 часов
- Учитывает комиссии при расчёте

## Как запустить

1. Установить Python 3.11+
2. Клонировать репозиторий и установить зависимости:
   ```bash
   git clone https://github.com/nikegor7/polymarket_bot.git
   cd polymarket_bot
   pip install -r requirements.txt
   ```
3. Создать файл `.env` с API-ключами (минимум: `ANTHROPIC_API_KEY`, `GNEWS_API_KEY`)
4. Запустить: `python main.py`

Подробная конфигурация — в [README.md](README.md) (англ.).

## Дисклеймер

Бот создан для исследовательских и образовательных целей. Торговля на рынках предсказаний несёт риск потери денег. По умолчанию бот работает в режиме симуляции (DRY RUN).
